# Drip Store

A Node.js backend project with API routes and basic server setup.

## 🚀 Tech Stack
- Node.js
- Express

## 📁 Project Structure
- `server.js` → Main server entry point  
- `routes/` → API route handlers  
- `models/` → Data models  
- `public/` → Static files  

## ▶️ Getting Started

### 1. Install dependencies
```bash
npm install
### 2. Start the server
```bash
npm start
```
